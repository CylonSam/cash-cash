# Cash Cash
This project is a finance api built with :heart: and Go.

## Tools

- Go
- Echo
- Gorm
- Postgres
- Docker

## Getting started

1. **Setup database**
    
    How: change to deployments folder and do: `docker compose up -d`

2. **Run project**

    How: from the root of the project, do: `go run cmd/main/main.go`
